NewFUHI 埋め込みサンプル

このページは WordPress 等に埋め込む前の動作確認用です。

1. 予約カレンダー

WordPress ショートコード: [newfuhi_booking store_id="1" api_key="YOUR_API_KEY"]

2. 本日のシフト

WordPress ショートコード: [newfuhi_shift store_id="1" api_key="YOUR_API_KEY"]

3. HTML 直接埋め込み（WordPress 不要の場合）

<iframe
  src="https://timebaibai.com/embed/booking/1/?api_key=YOUR_API_KEY"
  width="100%"
  height="600"
  style="border: none; max-width: 100%;"
  loading="lazy"
  title="予約カレンダー"
></iframe>

設定手順

  1. Timebaibai 管理画面 → メインページ設定 → 「外部埋め込みを有効化」を ON
  2. 店舗管理 → 対象店舗 → 「外部埋め込み」セクション → 「APIキー生成」アクション実行
  3. 生成された API キーを YOUR_API_KEY に置き換え
  4. （任意）「埋め込み許可ドメイン」に WordPress サイトのドメインを入力
